### 플러터란
플러터는 앱을 쉽게 만들수 있게 해주는 개발 도구 (ui 프레임워크)이다
### 주요 특징 
한번 개발 --> 여러 플랫폼 실행<br>
빠른 개발 속도<br>
디자인 자유도 높음<br>
Dart 언어 사용
### 플러터로 만들 수 있는것 
스마트폰 앱<br>
웹 사이트<br>
데스크톱 앱 (pc)
### 플러터 많이 쓰는 이유
개발 시간 줄어듬<br>
유지보수 쉬움<br>
스타트업에서 인기 많다<br>
### 플러터랑 네이티브의 차이점
#### 네이티브란
특정 운영체제에 맞게 직접 만든 앱을 의미한다
### 네이티브의 특징 
성능이 가장 좋음<br>
카메라,GPS, 센서 등 기능을 완전히 활용 가능<br>
각 기기에 최적화 
### flutter와 네이티브의 차이점
flutter 한번 코딩 --> android + ios 둘다 실행<br>
네이티브 --> 각각 따로 개발
#### 개발 속도
flutter --->  빠름 
네이티브 ---> 느림 
### 기능사용
flutter는 일부 기능을 플러그인 필요
네이티브는 모든 기기 기능 바로 사용가능 
| 항목   | Flutter | 네이티브  |
| ---- | ------- | ----- |
| 개발   | 1번      | 2번    |
| 성능   | 좋음      | 최고    |
| 속도   | 빠름      | 느림    |
| 기능   | 일부 제한   | 완전 자유 |
| 유지보수 | 쉬움      | 복잡    |
